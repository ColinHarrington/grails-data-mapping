Package: grails.http.client.test

[Groovy] Class TestRxHttpClientBuilder

      • grails.http.client.test.TestRxHttpClientBuilder

      • Allows for mocking and testing usages of RxHttpClientBuilder in application code

        Below is an example: 

           TestRxHttpClientBuilder client = new TestRxHttpClientBuilder()
   client.expect {
       uri '/foo/bar'
       method "POST"
       contentType 'application/json'
       json {
           title "Ping"
       }
   }.respond {
       created()
       json {
           title "Pong"
       }
   }

    Observable p = client.post("https://localhost:8080/foo/bar") {
        contentType 'application/json'
        json {
            title "Ping"
        }
    }

    assert client.verify()

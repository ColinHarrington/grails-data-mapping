    • All Implemented Interfaces and Traits:
      java.util.Collection

      public abstract class DirtyCheckableAwareCollection
extends java.lang.Object

      A java.util.Collection implementation setting it's owner's dirty status upon modification of the set. Internally org.grails.datastore.mapping.dirty.checking.DirtyCheckable is used which is applied to all domain classes using an AST transformation.
